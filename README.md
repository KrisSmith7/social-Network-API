# social-Network-API

  ## Description
   This project is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. It uses Express.js for routing, a MongoDB database, and the Mongoose ODM.

  ## Installation 
  To use, clone the repo, and run `node server` or `npm start` to start the server file. You can test routes by using Insomnia.

  Link to Github Repo: [https://github.com/krissmith7/social-network-api](https://github.com/krissmith7/social-network-api)
